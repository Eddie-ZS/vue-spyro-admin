### 📑 本地开发

> ⚠️ 本地开发需要保证 `nodejs 18`、`pnpm 9` 以上

```
git clone https://github.com/xiaoyuan-zs/vue-spyro-admin.git

cd spyro-vue-admin

pnpm install

pnpm dev

```
